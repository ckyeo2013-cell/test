# Multiplayer 2D Shooter

A simple 2D shooter with jumping, aiming, and multiplayer support.

## How to Play
1. Go to: [https://YOUR_GITHUB_USERNAME.github.io/REPO_NAME/](https://YOUR_GITHUB_USERNAME.github.io/REPO_NAME/)
2. The first person to open the page will get a link — share that link with friends so they can join.
3. Controls:
   - **WASD** = Move & Jump
   - **Mouse** = Aim
   - **Left Click** = Shoot
4. Kill all enemies to win!

## Features
- Multiplayer (no server needed, works directly in browser)
- Gravity & jumping
- Enemies & shooting
- Win screen when all enemies are dead
