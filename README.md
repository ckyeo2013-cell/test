# Multiplayer 2D Shooter

Play with friends online — **no server needed**, works on GitHub Pages.

## How to play
1. Open the page: https://YOUR_NAME.github.io/YOUR_REPO/
2. Click **CREATE ROOM** → you get a **4-digit code**.
3. Tell your friends to open the same link, type your code, and click **JOIN ROOM**.
4. You are now in the same game!

### Controls
- **WASD** = Move & Jump
- **Mouse** = Aim
- **Left Click** = Shoot
- Kill all enemies to win.

✅ Everyone sees each other move, jump, aim, and shoot.
✅ Works on any device with a browser.
